# Alex Gimourginas Personal Site

Personal portfolio site built with React + TypeScript + Vite and deployed via GitHub Pages.

## Tech Stack
- React
- TypeScript
- Vite
- GitHub Pages (deployment)

## Project Rules
- React + TypeScript only
- Use `react-router-dom` for routing
- Pages go in `src/pages`
- Reusable components go in `src/components`
- Data goes in `src/data`
- Avoid adding new libraries unless necessary
- Prefer small components over large files
- Keep styling simple and consistent

## Development

Run locally:

npm install  
npm run dev

Site runs at:
http://localhost:5173

## Deployment

Every push to `main` automatically deploys via GitHub Actions to:

https://alexgimourginas.github.io
